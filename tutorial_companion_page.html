Core Concepts

What this tutorial covers

Four essential ideas, explained from scratch with no prerequisites beyond high school math.

⚛️
Superposition & Entanglement
Qubits differ fundamentally from classical bits. Understand how quantum states represent exponential information.
🧪
Molecular Hamiltonians
The Hamiltonian encodes a molecule's full energy. Learn how atomic coordinates become qubit operators via Jordan-Wigner.
🤖
VQE + AI Optimizer
The Adam optimizer — same as in deep learning — drives the variational loop. Quantum ML in action.
💊
Drug Discovery Link
Ground state energy predicts molecular binding. Understand why this matters at the scale of real proteins.
Algorithm Walkthrough

VQE, step by step

01
Define molecular geometry
Specify element symbols and 3D atomic coordinates. For H₂: two hydrogen atoms 0.74 Å apart on the z-axis.
02
Build the qubit Hamiltonian
PennyLane runs Hartree-Fock, then maps fermionic operators to Pauli strings via Jordan-Wigner transform.
03
Design the ansatz circuit
Build a parameterized quantum circuit with Single and Double Excitation gates. Each gate has a trainable angle θ.
04
🤖 Run the AI optimizer (Adam)
The Adam optimizer (ML algorithm) computes gradients via the parameter-shift rule and updates all θ values to minimize energy.
05
Measure & repeat until convergence
The quantum circuit measures ⟨ψ(θ)|H|ψ(θ)⟩. Iterate until ΔE < 10⁻⁶ Hartree.
06
Interpret and extend
Plot the convergence and potential energy surface. Compare to exact FCI. Try the same code on LiH, H₂O, or NH₃.

# 🤖 AI-Powered: Adam optimizer (same as deep learning)
import pennylane as qml
from pennylane import qchem

# 1. Build molecular Hamiltonian from geometry
H, num_qubits = qchem.molecular_hamiltonian(
    ['H', 'H'], coordinates, basis='sto-3g'
)

# 2. Parameterized quantum circuit (ansatz)
dev = qml.device('default.qubit', wires=num_qubits)

@qml.qnode(dev)
def vqe_circuit(params, hamiltonian):
    qml.BasisState(hf_state, wires=range(num_qubits))
    for i, exc in enumerate(doubles):
        qml.DoubleExcitation(params[i], wires=exc)
    for i, exc in enumerate(singles):
        qml.SingleExcitation(params[len(doubles)+i], wires=exc)
    return qml.expval(hamiltonian)  # ← QUANTUM MEASUREMENT

# 3. AI optimization loop
opt = qml.AdamOptimizer(stepsize=0.02)  # same as neural nets!
params = np.zeros(num_params, requires_grad=True)

for _ in range(100):
    params, energy = opt.step_and_cost(
        vqe_circuit, params, hamiltonian=H
    )
# → Final energy: -1.13618819 Hartree  ✅ Chemical accuracy!
